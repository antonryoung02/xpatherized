# xpatherized README

This is the README for the extension "xpatherized". It provides format on save, copy minified xpath, and syntax coloring features for .xpath files.

Commands:

Command Palette > Minify XPath

- Saves the minified version of your .xpath file to clipboard

# Development

- npm compile
- run debugger on `extension.ts`

- update package.json
- npm run publish
